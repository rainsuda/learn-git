# git add
`git add` is a command that can add files to git version control.

for example:
```bash
$ vim git-add.md
$ git add git-add.md
```

Now we got:
```bash
$ git status
On branch master
Your branch is up-to-date with 'origin/master'.

Changes to be committed:
  (use "git reset HEAD <file>..." to unstage)

	new file:   git-add.md

```
